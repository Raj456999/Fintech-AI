# 💰 FinTech AI Assistant

An intelligent **FinTech-powered AI assistant** designed to simplify financial understanding, answer finance-related queries, and provide insights using data-driven retrieval and generative AI.

#Check this out:
https://fin-assist-agent.streamlit.app/

---

## 🚀 Overview

This project is a **Retrieval-Augmented Generation (RAG)-based FinTech assistant** that combines:
- Financial domain knowledge (RBI guidelines, fintech concepts, etc.)
- Document-based retrieval
- LLM-powered response generation

It acts as a smart chatbot that can answer questions related to:
- Banking & Digital Payments
- Loans & Credit systems
- UPI & Wallet ecosystems
- RBI regulations & financial policies
- General personal finance concepts

---

## 🧠 Key Features

- 📄 **Document-based Q&A (RAG system)**
- 🔍 Semantic search over financial documents
- 🤖 LLM-powered intelligent responses
- 📊 Finance domain knowledge integration
- ⚡ Fast retrieval using vector embeddings
- 🧾 Supports PDF / text-based financial data ingestion
- 💬 Chat interface for user interaction

---

## 🏗️ Tech Stack

- Python 🐍
- LangChain / LlamaIndex (RAG pipeline)
- FAISS / ChromaDB (Vector Database)
- OpenAI / HuggingFace Embeddings
- Streamlit (UI)
- PyPDF / Unstructured (Document parsing)

---

## 🔄 System Architecture
User Query
↓
Embedding Model
↓
Vector Database (PINECONE)
↓
Top-K Relevant Documents
↓
LLM (Response Generation)
↓
Final Answer
